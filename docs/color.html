颜色名mainv0.1.5 - Developing

有时在网站开发中会遇到这样的问题：
需要一个主题下所有颜色随着主题改变而改变，但在使用JavaScript改变时有时需要上百行代码。颜色名main可以解决这个问题。

更换全局主题只需要2行JavaScript代码(在Chrome或Microsoft Edge浏览器下尝试按Ctrl+Shift+J将它敲到调试区里试一试！)： 

javascript:document.getElementById("main_color").classList.remove("blue")
javascript:document.getElementById("main_color").classList.add("danger") // or blue, warning, ok, info
相应的，在HTML中你想要更改的地方添加class： 
<element class="blue">
    <!-- ... -->
</element>
