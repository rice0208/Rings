<code> 与 <pre> v0.1.5

 在Rings中，<code> 标签与 <pre> 标签可以设置多种颜色。

第一种方式：通过直接添加class来更换

以 <code> 标签为例，我们有了这样一行代码：
<code>print('Hello World')</code>
输出元素：

print('Hello World')

但当我们向标签添加class时： 
<code class="danger">print('Hello World')</code>
输出元素：

print('Hello World')

第二种方式：通过更改父元素的main主题色来更换
相对来说，第一种方式比较适合单个元素，而第二种方式更适合一整块元素的属性设置。
还是以code标签为例： 
<div>
    <code>print('Hello World')</code>
</div>
输出同上。

当我们往父元素（即示例中的div）添加class时： 
<div class="info">
    <code>print('Hello World')</code>
</div>
输出元素：

print('Hello World')


*注1：code支持blue,danger,warning,danger,info和gray.
*注2：虽然这两种方法在code元素上都达到了目的，几乎没有区别，但是我们还是想强调一下一个小细节：当你选取元素中的内容时，你会发现code选取的颜色是父元素的main-3。因此第一种的颜色是blue-3，而第二种则是info-3，如果谁往里面放了 a 标签，也是一个样。关于它的原因我们将在后面介绍。
