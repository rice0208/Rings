按钮样式v0.1.5 - Deprecatedv0.2.0 - Developing

 互联网开发正在不断扁平化，平滑化——因此浏览器提供的按钮已经算是过时了。

Rings使用了一系列自定义按钮样式。
通过给元素添加class .btn-* 从而添加自定义按钮样式。

有以下样式可选:


   .btn.btn-blue
 .btn.btn-danger
.btn.btn-warning
     .btn.btn-ok
   .btn.btn-info
   .btn.btn-dark
   .btn.btn-gray
  .btn.btn-light

在v2.0.0以上的版本，有以下的新增按钮样式：
Secondary按钮
.btn.btn-secondary
.btn.btn-secondary-blue
.btn.btn-secondary-danger
.btn.btn-secondary-warning
.btn.btn-secondary-ok
.btn.btn-secondary-info
.btn.btn-secondary-dark
.btn.btn-secondary-gray
.btn.btn-secondary-light
按钮大小
.btn.btn-small
.btn /* normal */
.btn.btn-large
Angular按钮
.btn.btn-angular
.btn.btn-angular-top
.btn.btn-angular-bottom
.btn.btn-angular-left
.btn.btn-angular-right
按钮组
<div class="btngroup">
    <button class="btn btn-blue btn-angular-right">Hello</button>
    <button class="btn btn-light btn-angular">World</button>
    <button class="btn btn-light btn-angular-left">Rings!</button>
</div>
