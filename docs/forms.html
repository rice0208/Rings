表单控件v0.1.5 - Developing

 使用Rings可以搭配使用class，使得表单可以拥有很棒的样式。下面了解一下特殊表单控件的使用。 首先，你需要知道的是：浏览器提供的表单控件一般不方便使用样式。（输入框除外，不信你亲测一下），因此，我们需要额外添加元素<label>

复选框样式v0.1.5 - Developing

Rings复选框用法如下： 
<form>
    <div><b>请选择你的爱好:</b>
        <input class="custom-checkbox" id="check1" type="checkbox" value="" name="check"><label for="check1"><p>篮球</p></label>
        <input class="custom-checkbox" id="check2" type="checkbox" value="" name="check"><label for="check2"><p>滑雪</p></label>
        <input class="custom-checkbox" id="check3" type="checkbox" value="" name="check"><label for="check3"><p>跳伞</p></label>
        <input class="custom-checkbox" id="check4" type="checkbox" value="" name="check"><label for="check4"><p>骑行</p></label>
        <input class="custom-checkbox" id="check5" type="checkbox" value="" name="check"><label for="check5"><p>游泳</p></label>
    </div>
</form>

复选框与单选框都有颜色样式选择：与code元素相同，如果input元素或父元素添加了class.blue .danger .warning .ok .info，颜色将会改变。你可以现场按下F12试一试。

开关样式v0.1.5 - Developing

Rings的开关按钮基于复选框，用法是复选框+label。 
<div class="switch">
    <input class="custom-checkbox-switch" type="checkbox" id="switch1">
    <label for="switch1">
</div>
显示效果：

另外，Rings开关还有丰富的颜色样式可供选择。
第一种样式：直接添加基本颜色class
一共有八种class可选： .blue .danger .warning .ok .info .gray .dark .light
第二种样式：添加from颜色与to颜色来改变详细变色方式
在第一种方式的基础上，在class前添加“from”与“to”从而改变样式。 例如.from-danger.to-ok显示效果为：

输入框样式v0.1.5 - Developing

Rings的输入框改善了原本丑陋而又不好用的浏览器输入框。 
<input class="custom-text">
