其它没有提到的class

 我们还有未提到的一些基本class，虽然不在前面的主要特性中，但还是比较实用的：
.inline: 
.inline{display: inline-block;}
.monospace: 
.mono,.monospace,.mono *,.monospace *{font-family:@font-mono;}
.align-left,.align-right,.align-center:
这三个不说都知道。
.float-* 
.float-right,.float-right *{float: right;}
.float-left,.float-right *{float: left;}
text-*

.text-light-1 *,
.text-light,

}

.text-light-2 {

}
.text-normal *,

font-weight: 400 !important;
}
.text-bold *,

}
.text-bold-2 *,

}

.lowercase {

}
.uppercase *,
