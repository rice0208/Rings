向表格添加数据：.figurev0.1.5

当我们在表格中显示数据时，数据的比较非常重要，因此Rings使用了等宽字体(Monospace)以对齐文本。如果将.figure加入td元素，数据将会自动向右对齐并等宽(灵感来自Microsoft Excel)。


效果如下： 
<table class="table-striped">
    <tbody>
        <tr>
            <th class="cell-1"></th><th class="cell-6">Set A</th><th class="cell-6">Set B</th><th class="cell-6">Set C</th><th class="cell-6">Set D</th><th class="cell-6">Set E</th><th class="cell-6">Set F</th><th class="cell-6">Set G</th>
        </tr>
        <tr>
            <td class="cell-1">1</td><td class="cell-6 figure">29.01</td><td class="cell-6 figure">26.40</td><td class="cell-6 figure">29.30</td><td class="cell-6 figure">28.36</td><td class="cell-6 figure">26.45</td><td class="cell-6 figure">26.42</td><td class="cell-6 figure">26.48</td>
        </tr>
        <tr>
            <td class="cell-1">2</td><td class="cell-6 figure">27.70</td><td class="cell-6 figure">26.51</td><td class="cell-6 figure">25.43</td><td class="cell-6 figure">24.21</td><td class="cell-6 figure">27.91</td><td class="cell-6 figure">24.32</td><td class="cell-6 figure">27.52</td>
        </tr>
    </tbody>
</table>
输出元素：
Set ASet BSet CSet DSet ESet FSet G
129.0126.4029.3028.3626.4526.4226.48
227.7026.5125.4324.2127.9124.3227.52

表格颜色样式v0.1.5 - Developing

表格的颜色样式也与其它的元素一样。注意：目前仍不支持随着父元素改变而改变！
示例：<table class="table-danger"> <!-- ... --> </table>
输出：
Set ASet BSet CSet DSet ESet FSet G
129.0126.4029.3028.3626.4526.4226.48
227.7026.5125.4324.2127.9124.3227.52
当然还有其它样式，目前支持5种基本颜色：blue, danger, warning, ok, info.
